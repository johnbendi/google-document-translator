In order to run, Google Document Translator needs java to be installed in your system. Please install the latest version from http://www.java.com/en/download/manual.jsp.

Now, open console in your system, type "java" and press enter. If the console prints the usage and options, then your system is ready to run the software. If java is not installed properly, it will show command not found.

Now run Gdt.bat. If you are using win7/vista, please run it with admin privilege.

This project is still in its early phase of development and we expect some errors to be reported by the users. Any bugs/query/suggestion will be handled as soon as possible
